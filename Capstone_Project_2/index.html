Mondrian Painting

I created a digital rendition of a Mondrian painting using CSS Grid, replicating the iconic style of the Dutch artist Piet Mondrian. The project focuses on achieving a clean and minimalist design by arranging colored blocks within a structured grid layout. By leveraging the flexibility of CSS Grid, I was able to accurately capture the geometric abstraction and balance of Mondrian's compositions. This project demonstrates the power of modern web technologies in creating visually compelling art-inspired designs.
